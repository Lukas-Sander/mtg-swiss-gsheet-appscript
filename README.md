# mtg-swiss-gsheet-appscript
apps script for google sheets for managing an mtg even using swiss pairings

## current state
the script can (incompletely) generate pairings for matches based on standings. standings are currently calculated, but not with a real score.

## up next
current goal is to make the pseudo-static version of the apps script functional.

## in the future
- create fully dynamic version of the apps script, which generates all needed tables
- create tutorial for initial setup
